Konstruct 1.2

Konstruct is a HTML & CSS framework developed to assist in building awesome stuff on the internet.

Documentation

Documentation is a huge priority for this project. We don't have this in place yet, but will soon!

Getting Started

You can get Konstruct in two ways right now:

  • Download the latest release - v.1.2
  • Clone the Main Branch - https://github.com/traaidmark/Konstruct.git

You will need to install NodeJS and Sass. When you have both, simply run npm install to install grunt to your repo. You can then process sass & js files with grunt watch.

Versions

Konstruct is versioned under the Semantic Versioning guidelines.

Releases are numbered as follows:

major.minor.patch

  • Bug fixes and the like goes to patch
  • Features and additions goes to minor
  • Complete new versions with no backwards compatibility path goes to major. This is usually also a milestone release and is considered stable and complete.
